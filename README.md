🚀 Profile Management System - Frontend

A modern React frontend for the Profile Management System with JWT Authentication, Material UI, Profile Image Upload, Search, Pagination and CRUD operations.

⸻

✨ Features

* User Login
* JWT Authentication
* Profile CRUD
* Image Upload
* Search Profiles
* Pagination
* Material UI Components
* Responsive Design
* Axios API Integration
* Protected Routes

⸻

🛠 Tech Stack

* React
* Vite
* Material UI
* Axios
* JavaScript
* CSS

⸻

🏗️ Architecture

React UI
     │
Axios API Calls
     │
Spring Boot Backend
     │
MongoDB
